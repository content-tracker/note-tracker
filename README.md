
# note-tracker

[🔧クロスSNS分析用トラッキングハブ]

以下のSNSからnoteへの流入を計測し、
発信活動の改善につなげるためのの分析プロジェクト

🌐 Threads・X・TikTok・Instagram


## Goals

- SNSごとの流入数を把握する
- 投稿内容と流入の相関を分析する
- note読了率との関係を検証する


## Why
SNS運用では「どこから来たか」は見えても、
「どんな投稿が読者につながるか」は見えづらい。

このプロジェクトでは、
複数SNSからnoteへの流入データを蓄積し、

・どのSNSが読まれやすいか
・どのような発信が読者行動につながるのか

を検証しています⚗️ˎˊ˗
