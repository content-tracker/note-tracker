# note-tracker

X・TikTok・Threadsからnoteへの流入を計測するクロスSNS分析用トラッキングハブ📊ˎˊ˗

## Purpose

複数SNSからnoteへの流入を可視化し、発信活動の改善につなげる。

## Goals

- SNSごとの流入数を把握する
- 投稿内容と流入の相関を分析する
- note読了率との関係を検証する


## why
SNS運用では「どこから来たか」は見えても、「どんな投稿が読者につながるか」は分かりづらい。

このプロジェクトでは、複数SNSからnoteへの流入データを蓄積し、発信と読者行動の関係を検証する⚗️🫧
